BashOnWindows(wsl)的安装,卸载与重装

Posted Jun 18, 2019

前提

win10不是所有版本都能安装wsl,生成版本 < 16215的无法安装wsl，生成版本大于16215的还分两种情况，一种需要在cmd命令行安装，另一种情况需要在应用商店安装，具体分界线在哪，我不清楚。 查看生成版本： 进入powershell，输入winver命令，弹出框里显示的OS内部版本就是生成版本

首次安装

  1. 开始->设置->更新与安全->针对开发人员, 选择”开发人员模式“
  2. 控制面板->程序->启用或关闭Windows功能, 勾选适用于Linux的Windows子系统, 并重启（如果系统里看不到这个选项就是生成版本不够原因）
  3. cmd命令行输入bash命令进行安装, 中途需要选择一些安装选项(通常是一路y下去)（pps: 如果显示bash不是内部命令,则需要到应用商店安装）
  4. 需要创建账号, root账号需要修改密码后才可以使用(sudo passwd root)

卸载

  1. 保存家目录并卸载ubuntu, cmd下输入如下命令

    lxrun /uninstall

    windows会询问一些选项让你来确认, 如果要一次性执行下去, 可以使用以下命令人代替:

    lxrun /uninstall /y

  2. 完全卸载

    lxrun /uninstall /full

重装

lxrun /install

完全移除Win10上的bash工具

在卸载BashOnWindows的前提下还需要在控制面板->程序->启用或关闭windows功能里勾掉适用于Linux的Windows子系统 , 并保存设置

如果这篇教程没有帮到你，请查看官方文档

https://docs.microsoft.com/zh-cn/windows/wsl/about
